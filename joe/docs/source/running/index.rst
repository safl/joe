===================
 Running Testplans
===================

When running tests with **cijoe** one uses the command-line tool
``cij_runner``:

.. literalinclude:: cij_runner.out
   :language: bash

The essential inputs to provide to the tool are:

* An :ref:`sec-environment`; describing **where** to run.
* One or more :ref:`sec-packages-testplans`; describing **what** to run.
* An output directory to store the results aka the **testrun**
